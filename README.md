# kaimin PUZZLE 🐑

可愛らしい羊のキャラクター「kaiminちゃん」が主役のMatch-2パズルゲーム！
隣り合った同じ色のブロックをクリックして消していく、シンプルながら爽快感のあるゲームです。

## 特徴
- **Match-2 ルール**: 同じ色が2つ以上繋がっている場所をクリックしてブロックを消します。
- **kaiminブロック**: 羊のマーク「🐑」がついた特殊ブロック。クリックすると周囲のブロックを巻き込んで爆発します！
- **レインボーブロック**: どの色のブロックとも繋がるワイルドカード！
- **Dream Time（フィーバー）**: 連続してブロックを消し、5コンボに達すると「Dream Time」に突入！スコアが2倍になります。コンボが途切れる（3秒間消さない）と終了します。
- **パーフェクトクリア**: 画面上のすべてのブロックを消すとボーナススコア (+5000pts) を獲得！

## 遊び方 (ローカル環境での起動)

Node.jsがインストールされている環境であれば、誰でも簡単に起動して遊ぶことができます。

1. このリポジトリをクローンまたはダウンロードします。
2. ターミナル（コマンドプロンプト）でこのフォルダを開きます。
3. 依存関係をインストールします：
   ```bash
   npm install
   ```
4. 開発用サーバーを起動します：
   ```bash
   npm run dev
   ```
5. ターミナルに表示されるURL（例: `http://localhost:5173/`）をブラウザで開いてプレイ！
